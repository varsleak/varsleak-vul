360安全路由

用户体验改进计划

欢迎您加入“360安全路由用户体验改进计划”，为了改善产品的用户体验，360安全路由可能会根据需要对产品的各项功能使用情况进行统计，我们希望通过分析统计数据持续不断地提升产品操作体验、运行性能，改善功能设计等，并且推出对用户有帮助的创新功能。在统计时，我们只对产品自身的内容进行统计，绝不涉及用户的个人隐私数据。

一、本计划的加入与退出

用户可以根据自己的意愿选择是否加入本计划，可随时加入，也可随时退出。360安全路由器的管理页面留有体验计划的加入与退出入口。

二、产品升级统计

360安全路由产品更新需要连接服务器，检查程序更新信息，对固件自身进行升级更新，以增强抵御各种网络恶意攻击能力和改善您使用产品时的体验。为了能够区分并为不同设备提供正确可用的升级服务，需要向服务器上传固件版本号等信息。如当用户开启自动更新后，360安全路由器会自动连接服务器进行升级。

三、加入用户体验改进计划范围

我们希望能够通过对用户反馈的信息分析，尽最大的努力提升360安全路由器的用户体验，提高产品的易用性和系统稳定性。具体包括：

1. 能够及时更新到最新的固件版本，更准确更及时更彻底地抵御各种网络恶意程序的攻击和修复存在的已知缺陷；

2. 能够全兼容适配全国各地不同的网络服务供应商（ISP）提供的各种网络接入服务，如ADSL，光纤或局域网等网络接入；

3. 能够确保360安全路由器的管理客户端应用程序，如手机APP，WINDOWS执行程序获得更流畅的用户体验；

4. 能够优化360WiFi扩展器等其他智能家居设备连接360安全路由器的连接方式，新增或改善这些设备与360安全路由互动功能；

5. 能够优化360安全路由固件系统和插件系统，方便第三方互联网业务插件接入，给用户提供优质的互联网服务，如游戏加速，下载加速等功能；

四、参加用户体验改进计划不会侵犯您的隐私

360公司严格遵守遵照国家有关的法律法规，本计划对相关数据信息的收集是完全匿名的，同时我们不会将收集的信息与其他任何个人信息进行匹配存储。我们仅为了提高产品对用户的服务质量和效率而收集相关信息，并将这些信息用于努力改善软件体验。

根据产品的发展，360可能会对360安全路由产品的用户体验改进计划进行修改，修改后，我们会在产品或网页中显著的位置发布相关信息并明示用户，以便及时通知到用户。如果您选择继续使用360安全路由的服务，即表示您同意接受这些修改。
